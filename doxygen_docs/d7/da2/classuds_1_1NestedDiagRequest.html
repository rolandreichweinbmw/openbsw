uint16_t uds::NestedDiagRequest::getStoredRequestLength ( ::estd::slice< uint8_t const > const &  request) const
protectedvirtual

Get needed length of storage for a given request. The default implementation returns the size of the request. Overload this if storing the request consumes more or less bytes.

Note
While the request is stored within the same message buffer that can be used for the response it is crucial to think about memory consumption of the stored request

requestRequest to store
