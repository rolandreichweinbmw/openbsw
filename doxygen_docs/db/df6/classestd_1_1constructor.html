Detailed Description

template<class T>
class estd::constructor< T >

A template class that provides a portable way to construct objects from memory that is owned by a fixed-size container. This class makes it possible to add objects that do not have default constructors. It is used for all of the emplace methods.

Template Parameters
TThe type of object to construct.

Vector Example

class MyClass
{
public:
MyClass(int i);
};
void emplaceExample(estd::vector<MyClass>& v)
{
for(int i = 0; i < 10; ++i)
{
// call the MyClass(int) constructor
v.emplace_back().construct(i);
}
}

Warning
Be careful using this constructor class. It does not support any kind of move semantics, i.e. when copying an instance of constructor you will end up having two instances pointing to the same memory region. This is a compromise to have still efficient runtime.
