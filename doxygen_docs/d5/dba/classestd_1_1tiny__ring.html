Detailed Description

template<class T, size_t N>
class estd::tiny_ring< T, N >

A space optimized implementation for small ring buffers.

This ring buffer implementation has very low memory overhead (2 bytes) but is limited to rings of a maximum size of 255 elements. Also writing to a full ring buffer will overwrite old elements without notice.
