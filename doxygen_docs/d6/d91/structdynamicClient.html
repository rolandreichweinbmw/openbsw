Detailed Description

template<typename T, typename IDynamicClient, int32_t Size, int32_t dynamicChannelsCount, int32_t dynamicClientsCount = 0>
struct dynamicClient< T, IDynamicClient, Size, dynamicChannelsCount, dynamicClientsCount >

This data type provides an abstract runtime mapping between a common shared I/O and higher-level instances that can use this I/O.

Usage: create an instance of dynamic client inside the class responsible for distribution of the I/O channels that have to be dynamic, and provide public methods for adding and removing the dynamic client.

Template Parameters
T- type that contains client's index + index of a channel inside a client.
IDynamicClient- type of an object that should be registered as a dynamic client
Size- bit width for count of clients, number of clients then is 2^Size
dynamicChannelsCount- number of dynamic channels that can be stored in this structure
dynamicClientsCount- number of dynamic clients that use this proxy.

This structure is responsible for distributing number of clients and number of instances per client. If T is 8 bit, then dynamicClient can devote N bits to instances' indexing and (8-N) bits to indexing inside a single client. Example: T = uint8_t, Size = 2 --> instanceNumber is in [0..3], numberInsideClient is in [0..31]
