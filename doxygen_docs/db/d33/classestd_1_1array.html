Detailed Description

template<class T, size_t N>
class estd::array< T, N >

Fixed-size sequence container that holds a specific number of elements ordered in a strict linear sequence.

[TPARAMS_START]

Template Parameters
TType of elements of array, aliased by typedef value_type.
NSize of the array provided as size_t. [TPARAMS_END]
