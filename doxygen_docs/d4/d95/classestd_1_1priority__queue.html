Detailed Description

template<class T, class Comparator = std::less<T>, class Container = ::estd::vector<T>>
class estd::priority_queue< T, Comparator, Container >

C++11 like implementation of a priority queue.

Template Parameters
Ttype of elements of the priority queue, aliased by typedef value_type.
ComparatorHow to compare the elements in the priority queue
ContainerUnderlying storage implementation of the priority queue.

Memory Usage

The memory usage in bytes for this priority_queue is determined by the underlying container. By default the stack uses a vector as the storage. This is done for performance.

When using the vector as the underlying container:

32 bit OS: N * sizeof(T) + 20
64 bit OS: N * sizeof(T) + 40

Note
It is possible there will be extra bytes used for proper memory alignment depending on your compiler settings.
