Detailed Description

Adapter class for an IReader, that forwards all read items to an IWriter.

Warning
This class will only work for single producer, single consumer use cases because the API is a two step API. Calling peek() on the source will call allocate() on the destination and calling release() will call commit(). If you have multiple producers, the integrator needs to take care that all calls are protected against each other correctly.
