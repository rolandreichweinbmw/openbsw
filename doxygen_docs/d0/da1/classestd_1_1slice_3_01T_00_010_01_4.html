Detailed Description

template<class T>
class estd::slice< T, 0 >

A wrapper around a pointer providing save access to a region of memory interpreted as a particular data type 'T'. This is a specialization of the slice<T,Size> template for the case Size == 0. Size being zero results in the size being managed at runtime. access at runtime.
