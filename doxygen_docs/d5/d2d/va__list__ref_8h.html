◆ va_list_ref

using estd::va_list_ref = typedef internal::_va_list_ref<va_list>

A simple helper class to store a reference to a va_list started with a call of macro va_start.

Note
In C++98 there's the lack of the va_copy macro and thus only simple references to va_list types can be used. Unfortunately the handling of these references is platform-specific. Therefore more than one implementation is provided and va_list_ref is a simple typedef
