Detailed Description

template<typename Counter, typename UserData>
class docan::DoCanJobHandle< Counter, UserData >

JobHandle consisting of 2 IDs: counter and user data. User data is used for uniquely identifying the job it belongs to. Counter is used for separating the same job IDs in a certain time frame.
