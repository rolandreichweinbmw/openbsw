Detailed Description

template<class T>
class estd::typed_mem< T >

Wrapper class that provides a memory area and lets the user emplace and instance of T in this memory at runtime. This class also erases the destructor call of T, i.e. if typed_mem goes out of scope, the destructor if the wrapped type will not be called. This can be done explicitly by calling destroy().
