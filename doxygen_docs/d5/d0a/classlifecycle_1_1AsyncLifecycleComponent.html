Detailed Description

A LifecycleComponent base class that allows to specify an async context for each transition to run in.

A component implementing AsyncLifecycleComponent must provide implementations for the three state transition functions init, run and shutdown. After registering a LifecycleComponent to the LifecycleManager, the LifecycleManager will make sure that these are executed at the appropriate time.
