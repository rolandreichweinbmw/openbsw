Detailed Description

template<class T, class Compare = default_compare<T>>
class util::estd::observable< T, Compare >

Implementation of the observer pattern (https://en.wikipedia.org/wiki/Observer_pattern)

An instance of this class maintains a list of observers that are functions to be called whenever the observed value changes.
