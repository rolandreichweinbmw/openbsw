Detailed Description

A simple StringWriter that allocates a temporary stream from a shared stream on construction and releases this stream on destruction.

Note
while output on shared streams is exclusive the returned output stream should be held only for a short while. Thus a shared string writer object typically will be instantiated from within a non-blocking function call.
