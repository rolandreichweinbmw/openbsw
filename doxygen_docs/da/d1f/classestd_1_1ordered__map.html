Detailed Description

template<class Key, class T, class Compare = std::less<Key>>
class estd::ordered_map< Key, T, Compare >

A C++ std::map like class implemented using a sorted vector.

keys

Two keys are considered equivalent if Compare returns false reflexively.

Template Parameters
KeyKey type of elements in this map.
TType of data elements mapped to a key.
CompareCompare function object for type Key.

Memory Usage

The memory usage in bytes for this ordered_map will be:

32 bit OS: N * (sizeof(Key) + sizeof(T)) + 20
64 bit OS: N * (sizeof(Key) + sizeof(T)) + 40

Note
It is possible there will be extra bytes used for proper memory alignment depending on your compiler settings.
