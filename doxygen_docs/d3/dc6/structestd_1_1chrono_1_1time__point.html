Detailed Description

template<typename Clock, typename Dur>
struct estd::chrono::time_point< Clock, Dur >

Class template estd::chrono::time_point represents a point in time. It is implemented as if it stores a value of type Duration indicating the time interval from the start of the Clock's epoch.

Template Parameters
clockClock, the clock on which this time point is measured
durationDuration, a estd::chrono::duration type used to measure the time since epoch
repRep, an arithmetic type representing the number of ticks of the duration
periodPeriod, a estd::ratio type representing the tick period of the duration
