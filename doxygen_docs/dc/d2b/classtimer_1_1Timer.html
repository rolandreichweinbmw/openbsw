Detailed Description

template<class LockGuard>
class timer::Timer< LockGuard >

A template class that serves as collection of timeouts. It manages and processes both cyclic and one-shot timeouts, providing functionality to set timeouts, check if a timer is active, and cancel the enqueued timeouts.

Template Parameters
LockGuardis the type that implements RAII-based lock for secure section.
